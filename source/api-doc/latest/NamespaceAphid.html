NamespaceAphid Class Reference

Inherits from JavaScript Object
Location aphid

Overview

The NamespaceAphid class represents the top namespace of OpenAphid. All classes and functions for OpenAphid specific features are managed inside this namespace.

Interfaces like console and XMLHttpRequest are in the global scope as they are in browsers.
